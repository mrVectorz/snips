This is a toolbox container to troubleshoot packet drops with mlx5 NICs.
It is based on fedora 43 base image.
It contains the tools and the collect script contained here.
- turbostat
- perf
- pcm
- dpdk
- dpdk-tools
- pciutils
- util-linux
- rdma-core
- libibverbs
- ethtool
- iproute
- procps-ng
- grep
- sed
- tar
- findutils

The hosting repo:
- quay.io/mmethot/fed-toolbox:f43
